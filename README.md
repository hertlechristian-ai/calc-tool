# calc-tool

Ein winziges Python-Übungsprojekt – ein einfacher Taschenrechner mit Tests.
Mein erstes Projekt zum Kennenlernen des GitHub-Workflows. 🚀

## Funktionen

- `add(a, b)` – Addition
- `subtract(a, b)` – Subtraktion
- `multiply(a, b)` – Multiplikation

## Verwendung

```bash
python calc.py
```

## Tests ausführen

```bash
python -m unittest -v
```
